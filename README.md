# 💰 Personal Expense Tracker

A simple and responsive web application that helps users manage their daily income and expenses.

## Features
- ➕ Add income and expense transactions
- 💰 Automatic balance calculation
- 📂 Category-based transactions
- 📅 Date selection
- 🔍 Search transactions
- 🗑 Delete transactions
- 🌙 Dark Mode
- 🥧 Expense Pie Chart (Chart.js)
- 💾 Data stored using Local Storage

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js
- Local Storage

## How to Run

1. Download or clone this repository.
2. Extract the ZIP file (if downloaded as ZIP).
3. Open `index.html` in any modern web browser (Google Chrome, Microsoft Edge, Firefox, etc.).
4. Start adding your income and expense transactions.
